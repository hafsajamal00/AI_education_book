# Introduction to ROS 2 for Humanoid Robotics

Welcome to the ROS 2 for Humanoid Robotics course! This comprehensive guide will help you understand ROS 2 as the middleware that connects AI agents (digital brain) to humanoid robot hardware and simulation.

## What You'll Learn

In this course on Physical AI & Humanoid Robotics, you will:

- Understand the fundamentals of ROS 2 and its importance for Physical AI
- Learn about ROS 2 architecture and the Data Distribution Service (DDS)
- Explore the differences between using ROS 2 in simulation versus with real humanoid robots
- Master the ROS 2 communication model including nodes, topics, and services
- Understand message passing and coordination mechanisms in humanoid robotics
- Learn Python integration using rclpy for humanoid robot control
- Explore robot structure and control basics specific to humanoid robots
- Work with URDF for humanoid robot models
- Learn how to bridge Python AI agents to humanoid robot controllers

## Target Audience

This course is designed for students with Python and basic AI knowledge who want to understand how ROS 2 serves as middleware connecting AI agents to humanoid robot hardware and simulation.

## Module Structure

This course is organized into the following modules:
