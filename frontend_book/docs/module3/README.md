# Module 3: The AI-Robot Brain (NVIDIA Isaac™)

Welcome to Module 3 of the Physical AI & Humanoid Robotics Learning Platform. In this module, you'll learn about NVIDIA Isaac, a comprehensive robotics platform that provides the AI brain for intelligent robots.

## What You'll Learn

- NVIDIA Isaac Sim fundamentals and architecture
- Isaac ROS perception and navigation packages
- Integration of AI models with robotic systems
- Nav2 navigation system for humanoid robots
- Bipedal and humanoid movement algorithms
- Perception systems for AI-powered robots

## Prerequisites

- Understanding of ROS 2 (from Module 1)
- Basic knowledge of digital twins (from Module 2)
- Familiarity with AI concepts

## Module Structure

- Chapter 1: NVIDIA Isaac Sim Fundamentals
- Chapter 2: Isaac ROS Perception & Navigation
- Chapter 3: Nav2, Bipedal & Humanoid Movement
